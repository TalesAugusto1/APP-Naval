## üöÄ Deployment Instructions

### For Reviewers: Testing with Expo Go

1. **Install Expo Go**:
   - iOS: Download from App Store
   - Android: Download from Google Play

2. **Run the app**:

   ```bash
   cd School-Manager
   npm install
   npm start
   ```

3. **Scan QR code**:
   - Open Expo Go and scan the QR code displayed in terminal
   - App will load with full functionality

### For Production: Building Standalone Apps

```bash
# Install EAS CLI
npm install -g eas-cli

# Login to Expo
eas login

# Build for Android
eas build --platform android --profile production

# Build for iOS
eas build --platform ios --profile production
```
